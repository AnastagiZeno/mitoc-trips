How does does the WSC approve trips?

To mark a trip as approved, use the trip management interface. Simply tick each trip as approved, and click submit.

How do trips lose approval?

If trip leaders edit the trip itself, they will lose its approval. Leaders are given a warning about this behavior if their trip is already approved.

Previously approved trips lose their approval when:

  • A trip leader edits any field of the trip. This includes the leader list, trip date, description, etc.

Trips do not lose approval if:

  • An admin or WSC member edits the trip
  • Leaders submit or modify trip itinerary
  • The participant list changes for any reason

How do we record lecture attendance?

Lecture attendance can always be manually set for any participant on the manage participants page.

A special page exists that allows participants to enter their login information and be marked as having attended lectures. Note that this does not actually log in the participants. The page is only viewable to the WSC - committee members are encouraged to bring laptops to lecture for sign-in. Alternatively, an admin can temporarily alter permissions to allow any participant to view the page.

How is the WIMP specified?

A comprehensive sheet for the WIMP is available at any time. It lists all the emergency information for all upcoming trips. This page is viewable only to members of the WSC and designated WIMPs. To specify a WIMP for a given week, have an admin add the participant to the WIMP group.
